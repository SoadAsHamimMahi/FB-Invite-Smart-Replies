Facebook sets personal limits to every person. Important to know:
1. The extension uses buttons' names to locate them, but sometimes the design of FB and names of buttons are changed. I am regularly updating the extension, if some functionality stops working - write me an email, usually, I update it on the same day :)
2. When you launch the extension - Chrome tries to update it. If there is a new version - it updates it even if the extension is running. So you may see 'undefined' text and clicks are not working. Just reload the page and try again.
3. Posts with a lot of reactions: extension automatically clicks on 'Load more' buttons, Facebook loads 50 'Invite' buttons at a time, but then start to load fewer buttons, that's why extension takes more time for such posts, but don't worry, everything is automatic ;)
