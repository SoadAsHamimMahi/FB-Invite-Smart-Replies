üêõ Common Issues & Solutions

Issue: "API keys not showing"

Solution: 1. Make sure you're in the Settings tab 2. Click "Add Key" and fill all required fields 3. Check browser console for errors 4. Verify chrome.storage.sync permissions

Issue: "Buttons opening new tabs"

Solution: 1. Ensure you're using the new popup.html (not the old version) 2. Check that popup.js is the integrated version 3. Verify manifest.json has correct popup configuration

Issue: "Extension not working on Facebook"

Solution: 1. Make sure you're on facebook.com domain 2. Check that manifest.json has correct host_permissions 3. Verify content scripts are injected properly 4. Check browser console for permission errors
