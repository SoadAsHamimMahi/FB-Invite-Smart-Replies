Additional info:
1. This tool uses Facebook's button names and UI elements to function. Facebook regularly updates its design, which may cause features to stop working. If something breaks, please report it to your support channel (or modify the code yourself if you're able).
2. When you launch the extension, Chrome might try to update it. If a new version of *your* extension is available, it will update. If you notice unexpected behavior, try reloading the Facebook page.
3. **Full scan or Real time functions stops unexpectedly?** If running on multiple PCs with Chrome Sync enabled, ensure consistent settings to avoid conflicts.
4. **Posts with a lot of reactions:** This extension automatically clicks 'Load more' buttons. Facebook may limit the number of reactions loaded to preserve server performance. We suggest you run the script every X days to process new reactions.
